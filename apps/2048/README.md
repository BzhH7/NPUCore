# 2048 Game

经典2048数字合并游戏，用于演示操作系统内核的终端I/O能力。

## 游戏规则

- 使用方向键或WASD移动所有方块
- 相同数字的方块碰撞会合并
- 每次移动后会随机出现一个新方块(2或4)
- 目标是合并出2048！

## 内核功能演示

此游戏展示了以下内核能力：

1. **termios 终端控制**
   - 原始模式读取
   - 禁用回显

2. **ANSI 转义序列**
   - 256色背景渲染
   - 光标控制

3. **输入处理**
   - 方向键转义序列解析 (`\033[A/B/C/D`)

4. **随机数生成**
   - `rand()` / `srand()`
   - `gettimeofday()` 作为种子

## 编译

```bash
make        # 编译两个架构版本
make clean  # 清理
```

## 控制

- `W` / `↑` - 向上移动
- `S` / `↓` - 向下移动
- `A` / `←` - 向左移动
- `D` / `→` - 向右移动
- `R` - 重新开始
- `Q` - 退出游戏
