# Snake Game (贪吃蛇)

经典贪吃蛇游戏，用于演示操作系统内核的终端I/O能力。

## 功能特性

- 经典贪吃蛇玩法
- WASD 和方向键支持
- 分数和速度系统
- 游戏结束后可重新开始

## 内核功能演示

此游戏展示了以下内核能力：

1. **termios 终端控制**
   - `tcgetattr()` / `tcsetattr()` - 终端属性读写
   - 原始模式 (raw mode) - 禁用行缓冲和回显

2. **文件描述符控制**
   - `fcntl()` - 设置非阻塞 I/O

3. **时间相关系统调用**
   - `gettimeofday()` - 获取当前时间
   - `usleep()` - 微秒级休眠

4. **标准 I/O**
   - `read()` - 非阻塞读取键盘输入
   - `printf()` - 输出 ANSI 转义序列

## 编译

```bash
make        # 编译两个架构版本
make clean  # 清理
```

## 运行

将编译后的二进制文件复制到文件系统镜像中，然后在内核 shell 中运行：

```
/snake
```

## 控制

- `W` / `↑` - 上移
- `S` / `↓` - 下移  
- `A` / `←` - 左移
- `D` / `→` - 右移
- `Q` - 退出游戏
- `R` - 游戏结束后重新开始
